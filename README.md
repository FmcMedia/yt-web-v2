# yt-web

A Dockerized web UI for downloading **DRM-free** YouTube content using yt-dlp.

> **Important**: This tool is intended only for content you have the legal right to download. Downloading videos may violate YouTube's Terms of Service. DRM-protected content (many movies, premium videos) is explicitly not supported.

## Quick Start

```bash
cd yt-web
docker compose up --build
```

Then open http://localhost:8080
